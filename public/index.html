      Vaccine Alert

      A new Vaccine Alert feature has been integrated in to Vaccine Bell. This feature is meant for those users who don't want to register in Vaccine Bell, but still need to receive vaccine notification. To make use of this, you need to navigate to Vaccine Alert page from Vaccine Bell application. There , an option is provided to choose the cities for which you want to get notified about the vaccine availability. Once it's done, you will get notified via the browser whenever vaccine is available at the selected cities, until the browser tab is closed. You will still receive notifications even if the browser is running in the background. This feature is available for both the PC and mobile phone
